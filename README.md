# Furry Hub Bot

Telegram-бот для просмотра фурри-контента из Reddit, Telegram и Twitter.

## Возможности

- `/start` — запуск бота
- `/random` — случайное изображение
- `/add` — добавить новый источник
- `/sources` — список источников
- `/parse` — запустить парсинг вручную

## Установка

1. Клонируй репозиторий
2. Создай файл `.env`:
